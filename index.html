Scroll Animated

Simple and easy way for animate a elements when your page scroll.

Usage

Include the stylesheet on your document's <head>
Scroll Animated use a style with animations based in Animate.CSS 

					
						
							<link rel="stylesheet" href="scrollAnimated.css">

Include the javascript on your documentâ€™s 

					
						<script src="/scrollAnimated.js" type="text/javascript"></script>

Initialize Scroll Animated 

					
						var myElement = new ScrollAnimated('.myElement', 'effect', duration, delay);

Contributing

Pull requests are the way to go here. All changes should be committed to src/ files. I only have two rules for submitting a pull request: match the naming convention (camelCase, categorised [fades, bounces, etc]) and let us see a demo of submitted animations in a pen. That last one is important.
